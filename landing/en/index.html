WHY IT MATTERS

Most "cleaners" sell fear. We don't.

You know the act: racing bars, invented "threats", one huge button and a subscription at the end. Lots of ritual, little truth. ZodHub CleanPC does the opposite: it shows you what's there, what's taking up space and what you can do — and only acts when you confirm.

The usual theatre

  • Animations and "scans" that fake working.
  • Exaggerated "threats" and "errors" to scare you.
  • Data sent off to someone else's servers.
  • Pricey subscription for what should be free.

What ZodHub CleanPC does

  • Shows real system figures, measured in Rust.

FEATURES

The essentials, done right.

No filler features or modules you'll never use. Each tool solves something concrete — and all the heavy lifting runs in Rust, on your machine.

Cache cleanup

Finds app and system caches, tells you how much they take, and frees space only when you confirm.

Free up memory

Eases memory pressure when the machine is loaded. We flag approximations as such, without overstating.

.DS_Store

Finds and tidies the .DS_Store files macOS scatters everywhere, for cleaner folders.

Network radar

Visualizes connections, listening ports and network activity with real system data. A representation, not decoration.

Automatic tasks

Schedule periodic clean-ups with launchd. Set it once and forget it; it runs in the background.

System monitor

Live CPU, memory and processes, Activity-Monitor style, to see what's really going on.

SECURITY & ENGINE

All the heavy lifting runs in Rust, on your machine.

Enumerating, measuring, deleting, compressing, scanning the network: it's all done with safe native code and without freezing the interface. And with strict rules so it never touches what it shouldn't.

Validates that every path is within allowed scopes.
Never follows symbolic links when calculating sizes.
Never touches system paths protected by SIP.
Requests admin privileges only when needed.
